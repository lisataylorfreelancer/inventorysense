# Template Selection

AI Chat Application with Convex Backend

Use when:
- Building AI chat applications
- Applications requiring conversational AI features
- Chat interfaces with message history
- AI-powered assistants or chatbots
- Applications that need OpenRouter API integration

Avoid when:
- Simple static apps without AI features
- Apps that don't need chat functionality
- Client-side only apps

Built with:
- React Router, ShadCN UI, Tailwind, Lucide Icons, ESLint, Vite
- Andromo Convex Backend with AI chat integration
- Convex Auth for authentication
- OpenRouter API for AI model access
